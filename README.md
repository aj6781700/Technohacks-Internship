# Exploratory Data Analysis on Netflix Dataset

## 📌 Objective
Perform exploratory data analysis on Netflix Movies and TV Shows dataset using Python libraries.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Analysis Performed
- Data Cleaning
- Missing Value Handling
- Movies vs TV Shows Analysis
- Top Countries Analysis
- Ratings Distribution
- Release Year Trend
- Genre Analysis

---

## 📈 Key Insights
- Netflix contains more Movies than TV Shows.
- United States produces the highest amount of content.
- TV-MA is the most common rating.
- Content production increased rapidly after 2015.

---

## 📂 Dataset
Netflix Movies & TV Shows Dataset from Kaggle.

# House Price Prediction using Linear Regression

## Objective
Build a Linear Regression model to predict house prices using housing dataset features.

## Dataset
Housing Prices Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Steps Performed
1. Loaded the dataset
2. Checked dataset information and missing values
3. Performed basic statistical analysis
4. Created visualizations
5. Converted categorical columns into numerical values
6. Split data into training and testing sets
7. Trained Linear Regression model
8. Evaluated model performance

## Visualizations
- Price Distribution
- Area vs Price Scatter Plot
- Correlation Heatmap
- Actual vs Predicted Prices

## Evaluation Metrics
- Mean Absolute Error
- Mean Squared Error
- Root Mean Squared Error
- R2 Score

## Conclusion
The Linear Regression model was successfully trained to predict house prices.  
The project helped in understanding regression modeling, preprocessing, visualization and evaluation.
