# Exploratory Data Analysis on Netflix Dataset

## 📌 Objective
Perform exploratory data analysis on Netflix Movies and TV Shows dataset using Python libraries.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

---

## 📊 Analysis Performed
- Data Cleaning
- Missing Value Handling
- Movies vs TV Shows Analysis
- Top Countries Analysis
- Ratings Distribution
- Release Year Trend
- Genre Analysis

---

## 📈 Key Insights
- Netflix contains more Movies than TV Shows.
- United States produces the highest amount of content.
- TV-MA is the most common rating.
- Content production increased rapidly after 2015.

---

## 📂 Dataset
Netflix Movies & TV Shows Dataset from Kaggle.
